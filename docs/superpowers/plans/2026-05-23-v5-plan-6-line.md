# Plan 6: Line Sketch + Sketch Scaffolding Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the sketch infrastructure pattern (marker components, OnEnter/OnExit lifecycle, cleanup helpers, navigation wiring) and the first concrete sketch — Line — that uses it: WGSL compute kernel for ~5K particles attracted to the pointer, rendered as instanced quads, with `LineSettings` registered through the Plan 5 derive macro. No CPU fallback (per the 2026-05-23 spec amendment). Audio coupling and heatmap-image spawn deferred to Plan 7.

**Architecture:** The sketch pattern is "plugin per sketch + marker component per sketch." Each sketch's plugin (`LinePlugin`, future `FlamePlugin`, etc.) registers its settings, schedules its systems on `Update.run_if(in_state(AppState::Line).and(in_state(SketchActivity::Active)))`, spawns its entities under a marker (`LineRoot`) on `OnEnter(AppState::Line)`, and despawns them on `OnExit(AppState::Line)` via a generic `despawn_with::<LineRoot>` helper in `wc-core`. Line itself uses a WGSL compute pipeline owned by a custom render graph node: a `ShaderStorageBuffer<[Particle; N]>` is updated each frame by `assets/shaders/line/simulate.wgsl`, then rendered as instanced quads through a `Material2d` whose vertex shader reads the same buffer. The pointer attractor uses Plan 3's `PointerState` to push a sim-uniform buffer (`SimParams { dt, gravity, attractor_pos, drag }`) each frame.

**Tech Stack:** Rust 1.89, Bevy 0.18.1 (`bevy_render::storage::ShaderStorageBuffer`, `render_graph::Node`, `Material2d`), `bytemuck` for `Pod`/`Zeroable` derives on GPU structs, existing `leafwing-input-manager` 0.20 / `bevy_egui` 0.39 / `wc-core-macros` from prior plans.

```rust
/// System that, once per frame, walks the registry calling each entry's
/// `diff_requires_restart_fn` and emits a [`SketchRestart`] if any returned
/// true.
///
/// Snapshots only the function pointer + storage key for each entry into a
/// stack `SmallVec` so the registry resource stays unborrowed while the
/// per-type functions reborrow the world. No per-frame `Vec<SettingDef>`
/// clone; the `def` field is left untouched.
pub fn emit_restart_events(world: &mut World) {
    use smallvec::SmallVec;
    // Snapshot the (fn, key) pairs. Most apps will register ≤8 settings types.
    let snapshot: SmallVec<[(fn(&mut World) -> bool, &'static str); 8]> = world
        .get_resource::<SettingsRegistry>()
        .map(|r| {
            r.entries
                .iter()
                .map(|e| (e.diff_requires_restart_fn, e.storage_key))
                .collect()
        })
        .unwrap_or_default();
    for (diff_fn, storage_key) in snapshot {
        if diff_fn(world) {
            world
                .resource_mut::<bevy::prelude::Messages<SketchRestart>>()
                .write(SketchRestart { storage_key });
        }
    }
}
```

```rust
//! Curated user-facing settings panel.
//!
//! Walks [`super::SettingsRegistry`] each frame and, for each registered
//! settings resource, renders an `egui::CollapsingHeader` containing typed
//! widgets for every `SettingDef` whose `category == User`. Field values are

use bevy::reflect::{ReflectMut, ReflectRef, TypeRegistration};

/// Plugin assembly hook called by [`super::SettingsPlugin::build`].

pub(super) fn add_systems(app: &mut App) {

/// Exclusive system that draws the user panel.
fn draw_user_panel(world: &mut World) {
    // Skip when no egui context is up (e.g., MinimalPlugins test harness).
    if !world.contains_resource::<bevy_egui::EguiUserTextures>() {
        return;
    }
    // Snapshot the storage keys we need to iterate so the registry resource
    // stays unborrowed while we mutate per-type resources.
    let keys: Vec<&'static str> = world
        .get_resource::<SettingsRegistry>()
        .map(|r| r.entries.iter().map(|e| e.storage_key).collect())
        .unwrap_or_default();
    if keys.is_empty() {
        return;
    }

    let mut state: bevy::ecs::system::SystemState<EguiContexts<'_, '_>> =
        bevy::ecs::system::SystemState::new(world);
    let mut contexts = state.get_mut(world);
    let Ok(ctx) = contexts.ctx_mut() else {
        return;
    };
    let ctx = ctx.clone();
    state.apply(world);

    egui::Window::new("Settings")
        .id(egui::Id::new("wc-settings-user-panel"))
        .default_open(true)
        .show(&ctx, |ui| {
            for key in keys {
                render_section_by_key(world, ui, key);
            }
        });
}

/// Look up the type registration matching `storage_key` and render its
/// `User`-category fields. Walks the `TypeRegistry` to find the registered
/// settings type whose `SETTINGS_STORAGE_KEY` matches; uses reflection to
/// read/write fields without static type knowledge.
fn render_section_by_key(world: &mut World, ui: &mut egui::Ui, storage_key: &'static str) {
    // Find the entry's defs (still a borrow of the resource; copy out).
    let defs: Vec<SettingDef> = world
        .get_resource::<SettingsRegistry>()
        .and_then(|r| r.entries.iter().find(|e| e.storage_key == storage_key))
        .map(|e| e.def.clone())
        .unwrap_or_default();
    if defs.iter().all(|d| d.category != SettingsCategory::User) {
        return;
    }

    ui.collapsing(storage_key, |ui| {
        // Walk the type registry to find the settings type by its
        // SketchSettings::STORAGE_KEY. We compare by &'static str pointer
        // identity is unreliable; compare by value.
        let type_id = world
            .resource::<AppTypeRegistry>()
            .read()
            .iter()
            .find_map(|reg| settings_type_id_for_key(reg, storage_key));
        let Some(type_id) = type_id else {
            ui.label("(settings type not in TypeRegistry — register via App::register_type)");
            return;
        };

        // Get a Reflect handle on the resource.
        let registry = world.resource::<AppTypeRegistry>().clone();
        let registry_read = registry.read();
        let Some(type_data) = registry_read
            .get_type_data::<bevy::ecs::reflect::ReflectResource>(type_id)
        else {
            ui.label("(no ReflectResource on settings type)");
            return;
        };

        // Borrow the resource as &mut dyn Reflect and walk the user fields.
        let mut reflect = type_data.reflect_mut(world).map(|r| r);
        let Some(mut reflect) = reflect else {
            ui.label("(resource not present)");
            return;
        };
        render_user_fields_via_reflect(reflect.as_mut(), &defs, ui);
    });
}

/// Walk `reflect` (a `&mut dyn Reflect` over the settings struct) and render
/// each user-category field as a typed widget.
fn render_user_fields_via_reflect(
    reflect: &mut dyn Reflect,
    defs: &[SettingDef],
    ui: &mut egui::Ui,
) {
    use bevy::reflect::Struct;

    let ReflectMut::Struct(struct_mut) = reflect.reflect_mut() else {
        ui.label("(settings is not a struct)");
        return;
    };

    for def in defs.iter().filter(|d| d.category == SettingsCategory::User) {
        let Some(field) = struct_mut.field_mut(def.field_name) else {
            continue;
        };
        render_widget(field, def, ui);
    }
}

/// Render one widget into `field` based on the metadata in `def`.
fn render_widget(field: &mut dyn Reflect, def: &SettingDef, ui: &mut egui::Ui) {
    match &def.kind {
        SettingKind::Number(range) => render_number(field, def.label, range, ui),
        SettingKind::Boolean => render_bool(field, def.label, ui),
        SettingKind::Color => render_color(field, def.label, ui),
        SettingKind::Text => render_text(field, def.label, ui),
    }
}

/// Render a numeric field. Dispatches on the field's concrete Rust type
/// (u32, f32, etc.) via `try_downcast_mut`.
fn render_number(field: &mut dyn Reflect, label: &str, range: &super::def::NumberRange, ui: &mut egui::Ui) {
    let lo = range.min.unwrap_or(0.0);
    let hi = range.max.unwrap_or(1.0);
    let step = range.step;

    if let Some(v) = field.try_downcast_mut::<u32>() {
        let mut tmp = *v as i64;
        let mut slider = egui::Slider::new(&mut tmp, (lo as i64)..=(hi as i64)).text(label);
        if let Some(s) = step {
            slider = slider.step_by(s);
        }
        if ui.add(slider).changed() {
            *v = tmp.max(0) as u32;
        }
        return;
    }
    if let Some(v) = field.try_downcast_mut::<f32>() {
        let mut slider = egui::Slider::new(v, (lo as f32)..=(hi as f32)).text(label);
        if let Some(s) = step {
            slider = slider.step_by(s);
        }
        ui.add(slider);
        return;
    }
    if let Some(v) = field.try_downcast_mut::<f64>() {
        let mut slider = egui::Slider::new(v, lo..=hi).text(label);
        if let Some(s) = step {
            slider = slider.step_by(s);
        }
        ui.add(slider);
        return;
    }
    ui.label(format!("(unsupported number type for {label})"));
}

fn render_bool(field: &mut dyn Reflect, label: &str, ui: &mut egui::Ui) {
    if let Some(v) = field.try_downcast_mut::<bool>() {
        ui.checkbox(v, label);
    } else {
        ui.label(format!("(expected bool for {label})"));
    }
}

fn render_color(field: &mut dyn Reflect, label: &str, ui: &mut egui::Ui) {
    if let Some(v) = field.try_downcast_mut::<[f32; 4]>() {
        ui.horizontal(|ui| {
            ui.label(label);
            ui.color_edit_button_rgba_unmultiplied(v);
        });
    } else {
        ui.label(format!("(expected [f32; 4] for {label})"));
    }
}

fn render_text(field: &mut dyn Reflect, label: &str, ui: &mut egui::Ui) {
    if let Some(v) = field.try_downcast_mut::<String>() {
        ui.horizontal(|ui| {
            ui.label(label);
            ui.text_edit_singleline(v);
        });
    } else {

    }
}

/// Look up a TypeRegistration's `SketchSettings::STORAGE_KEY` and return its
/// `TypeId` if it matches.
///
/// The derive macro emits the storage key as a const associated to the
/// trait, not as type-registration metadata. We sidestep that by iterating
/// the registry and asking each type's `ReflectResource` whether its
/// SketchSettings impl has the matching key — but reflection doesn't carry
/// trait const data. Instead, the registry stores the (type_id, storage_key)
/// mapping at registration time. See `Task 4` for the registry change.
fn settings_type_id_for_key(
    reg: &TypeRegistration,
    storage_key: &str,
) -> Option<std::any::TypeId> {
    use super::registry::SettingsTypeKey;
    let data = reg.data::<SettingsTypeKey>()?;
    (data.0 == storage_key).then(|| reg.type_id())
}
```

- [ ] **Step 1: Create the autosave module**

```rust
//! Debounced settings auto-save.
//!
//! When any registered settings resource changes, schedule a save after a
//! short idle window. This keeps disk writes bounded (one write per debounce
//! window, not one per keystroke) while ensuring no edit is lost on shutdown
//! beyond the debounce-window worth of buffered changes.

use bevy::prelude::*;

use super::registry::SettingsRegistry;

/// Debounce window. Saves trigger this many seconds after the last change.
pub const DEBOUNCE_SECS: f32 = 0.5;

/// Per-storage-key debounce state. None = clean. Some(timer) = pending save.
#[derive(Resource, Default, Debug)]
pub struct AutosaveState {
    /// Map of storage_key → remaining seconds until save fires.
    pub pending: bevy::utils::HashMap<&'static str, f32>,
}

/// Detect resource changes and (re)start the per-type debounce timer.
/// Runs in `Update` after `emit_restart_events` so we observe the same
/// change ticks.
pub fn detect_changes(world: &mut World) {
    let snapshot: Vec<(fn(&World) -> bool, &'static str)> = world
        .get_resource::<SettingsRegistry>()
        .map(|r| {
            r.entries
                .iter()
                .map(|e| (e.is_changed_fn, e.storage_key))
                .collect()
        })
        .unwrap_or_default();
    let mut to_arm = Vec::new();
    for (changed_fn, key) in snapshot {
        if changed_fn(world) {
            to_arm.push(key);
        }
    }
    if to_arm.is_empty() {
        return;
    }
    let mut state = world.resource_mut::<AutosaveState>();
    for key in to_arm {
        state.pending.insert(key, DEBOUNCE_SECS);
    }
}

/// Advance debounce timers and fire `save_fn` when a timer reaches zero.
pub fn tick(world: &mut World) {
    let dt = world.resource::<Time>().delta_secs();
    let to_save: Vec<&'static str> = {
        let mut state = world.resource_mut::<AutosaveState>();
        let mut fire = Vec::new();
        state.pending.retain(|key, timer| {
            *timer -= dt;
            if *timer <= 0.0 {
                fire.push(*key);
                false
            } else {
                true
            }
        });
        fire
    };
    if to_save.is_empty() {
        return;
    }
    let snapshot: Vec<(fn(&World), &'static str)> = world
        .get_resource::<SettingsRegistry>()
        .map(|r| {
            r.entries
                .iter()
                .map(|e| (e.save_fn, e.storage_key))
                .collect()
        })
        .unwrap_or_default();
    for key in to_save {
        if let Some((save_fn, _)) = snapshot.iter().find(|(_, k)| *k == key) {
            save_fn(world);
            tracing::debug!(%key, "settings saved (debounce window elapsed)");
        }
    }
}
```

```wgsl
// Line particle simulation — one workgroup per 64 particles.
//
// Reads SimParams from a uniform buffer at @group(0) @binding(0).
// Reads + writes Particles in a storage buffer at @group(0) @binding(1).

struct Particle {
    position: vec2<f32>,
    velocity: vec2<f32>,
};

struct SimParams {
    dt: f32,
    drag: f32,
    attractor_pos: vec2<f32>,
    attractor_radius: f32,
    gravity_constant: f32,
    attractor_enabled: f32,
    _pad: f32,
};

@group(0) @binding(0) var<uniform> params: SimParams;
@group(0) @binding(1) var<storage, read_write> particles: array<Particle>;

@compute @workgroup_size(64)
fn main(@builtin(global_invocation_id) id: vec3<u32>) {
    let idx = id.x;
    let count = arrayLength(&particles);
    if (idx >= count) {
        return;
    }
    var p = particles[idx];

    // Gravity toward the attractor.
    if (params.attractor_enabled > 0.5) {
        let to_attr = params.attractor_pos - p.position;
        let dist_sq = max(dot(to_attr, to_attr), params.attractor_radius * params.attractor_radius);
        let inv_dist = 1.0 / sqrt(dist_sq);
        let force = params.gravity_constant * inv_dist * inv_dist;
        p.velocity = p.velocity + to_attr * inv_dist * force * params.dt;
    }

    // Apply drag.
    p.velocity = p.velocity * (1.0 - params.drag * params.dt);

    // Integrate.
    p.position = p.position + p.velocity * params.dt;

    particles[idx] = p;
}
```

```wgsl
// Line particle render — one instanced quad per particle.
//
// Vertex shader reads particle data from a storage buffer at
// @group(2) @binding(0) (group 2 = material group in Bevy's Material2d).

struct Particle {
    position: vec2<f32>,
    velocity: vec2<f32>,
};

@group(2) @binding(0) var<storage, read> particles: array<Particle>;

struct VertexInput {
    @location(0) local_pos: vec3<f32>,
    @builtin(instance_index) instance: u32,
};

struct VertexOutput {
    @builtin(position) clip_position: vec4<f32>,
    @location(0) brightness: f32,
};

@vertex
fn vertex(in: VertexInput) -> VertexOutput {
    let p = particles[in.instance];
    // Quad lives in unit-space [-0.5..0.5]; scale to a few pixels per
    // particle. World units flow through Bevy's view/projection in
    // Material2d's auto-binding.
    let scale = 1.5;
    let world_pos = vec3<f32>(p.position + in.local_pos.xy * scale, 0.0);
    var out: VertexOutput;
    out.clip_position = vec4<f32>(world_pos, 1.0);
    out.brightness = clamp(length(p.velocity) * 0.005, 0.05, 1.0);
    return out;
}

@fragment
fn fragment(in: VertexOutput) -> @location(0) vec4<f32> {
    let b = in.brightness;
    return vec4<f32>(b, b * 0.85, b * 0.6, 1.0);
}
```

```rust
    buffers: Res<'_, bevy::render::storage::RenderAssets<bevy::render::storage::GpuShaderStorageBuffer>>,
) {
    let Some(particle_buffer) = buffers.get(&sim.particles_handle) else {
        return;
    };
    // Write SimParams into a tiny GPU buffer this frame.
    let params_buffer = render_device.create_buffer_with_data(
        &bevy::render::render_resource::BufferInitDescriptor {
            label: Some("line_sim_params"),
            contents: bytemuck::bytes_of(&sim.params),
            usage: BufferUsages::UNIFORM | BufferUsages::COPY_DST,
        },
    );

    let bind_group = render_device.create_bind_group(
        "line_compute_bind_group",
        &pipeline.bind_group_layout,
        &[
            BindGroupEntry {
                binding: 0,
                resource: params_buffer.as_entire_binding(),
            },
            BindGroupEntry {
                binding: 1,
                resource: particle_buffer.buffer.as_entire_binding(),
            },
        ],
    );

    let dispatch_size = sim.particle_count.div_ceil(WORKGROUP_SIZE);
    commands.insert_resource(LineComputeBindGroup {
        bind_group,
        dispatch_size,
    });
}

/// Render graph node that dispatches the compute pipeline.
struct LineComputeNode;

impl render_graph::Node for LineComputeNode {
    fn run(
        &self,
        _graph: &mut render_graph::RenderGraphContext<'_>,
        render_context: &mut RenderContext<'_>,
        world: &World,
    ) -> Result<(), render_graph::NodeRunError> {
        let Some(bg) = world.get_resource::<LineComputeBindGroup>() else {
            return Ok(());
        };
        let pipeline_cache = world.resource::<PipelineCache>();
        let pipeline = world.resource::<LinePipeline>();
        let Some(compute_pipeline) =
            pipeline_cache.get_compute_pipeline(pipeline.pipeline_id)
        else {
            // Pipeline still compiling — try again next frame.
            return Ok(());
        };

        let mut pass = render_context
            .command_encoder()
            .begin_compute_pass(&ComputePassDescriptor {
                label: Some("line_compute_pass"),
                timestamp_writes: None,
            });
        pass.set_pipeline(compute_pipeline);
        pass.set_bind_group(0, &bg.bind_group, &[]);
        pass.dispatch_workgroups(bg.dispatch_size, 1, 1);
        Ok(())
    }
}
```

Adapt to the installed crate. The conceptual shape — extract resource into render world, prepare-bind-group system, compute node — is invariant.

```rust
//! Line sketch update systems.

use bevy::prelude::*;
use bevy::render::storage::ShaderStorageBuffer;
use wc_core::input::pointer::PointerState;

use super::compute::LineSimParams;
use super::material::LineMaterial;
use super::particle::{Particle, SimParams};
use super::settings::LineSettings;

/// Marker placed on every entity owned by the Line sketch. `OnExit` despawns
/// everything tagged with this marker.
#[derive(Component)]
pub struct LineRoot;

/// `OnEnter(AppState::Line)`. Allocates the particle buffer, spawns the
/// rendered entity, and installs `LineSimParams`.
pub fn spawn_line(
    mut commands: Commands<'_, '_>,
    settings: Res<'_, LineSettings>,
    mut buffers: ResMut<'_, Assets<ShaderStorageBuffer>>,
    mut materials: ResMut<'_, Assets<LineMaterial>>,
    mut meshes: ResMut<'_, Assets<Mesh>>,
) {
    let count = settings.particle_count.max(1);

    // Initial state: particles arranged in a square grid centered on origin.
    let side = (count as f32).sqrt().ceil() as u32;
    let spacing = 4.0;
    let mut initial = Vec::with_capacity(count as usize);
    for i in 0..count {
        let x = (i % side) as f32 - (side as f32 * 0.5);
        let y = (i / side) as f32 - (side as f32 * 0.5);
        initial.push(Particle {
            position: [x * spacing, y * spacing],
            velocity: [0.0, 0.0],
        });
    }

    let particles_handle = buffers.add(ShaderStorageBuffer::from(bytemuck::cast_slice::<Particle, u8>(&initial).to_vec()));

    let material = materials.add(LineMaterial {
        particles: particles_handle.clone(),
    });

    // Unit quad mesh, instanced `count` times by the render shader.
    let quad = meshes.add(Mesh::from(Rectangle::new(1.0, 1.0)));

    commands.spawn((
        LineRoot,
        Mesh2d(quad),
        bevy::sprite::MeshMaterial2d(material),
        // Bevy auto-attaches Transform / GlobalTransform when needed by the
        // 2D mesh bundle; spawn explicitly for clarity.
        Transform::default(),
        GlobalTransform::default(),
        Visibility::default(),
    ));

    commands.insert_resource(LineSimParams {
        params: SimParams::default(),
        particles_handle,
        particle_count: count,
    });
    tracing::info!(count, "spawned Line sketch");
}

/// `Update`-on-`sketch_active(Line)`. Pushes per-frame sim params using the
/// pointer state for the attractor and `LineSettings` for the constants.
pub fn update_sim_params(
    time: Res<'_, Time>,
    settings: Res<'_, LineSettings>,
    pointer: Res<'_, PointerState>,
    mut sim: ResMut<'_, LineSimParams>,
) {
    let attractor_enabled = if pointer.active { 1.0 } else { 0.0 };
    sim.params = SimParams {
        dt: time.delta_secs().min(0.05), // cap dt to avoid blow-up on stalls
        drag: settings.drag,
        attractor_pos: [pointer.position.x, pointer.position.y],
        attractor_radius: settings.attractor_radius,
        gravity_constant: settings.gravity_constant,
        attractor_enabled,
        _pad: 0.0,
    };
}
```

```rust
//! Line sketch — particles attracted to a pointer-driven gravity well.
//!
//! ## Data flow
//!
//! 1. `OnEnter(AppState::Line)` runs [`systems::spawn_line`]: allocates the
//!    particle storage buffer, spawns the entity tree under [`LineRoot`],
//!    and installs [`compute::LineSimParams`].
//! 2. Every `Update` while `sketch_active(AppState::Line)` is true,
//!    [`systems::update_sim_params`] writes the current pointer position +
//!    `LineSettings` values into `LineSimParams`.
//! 3. The render world extracts `LineSimParams` and dispatches the compute
//!    pipeline (`assets/shaders/line/simulate.wgsl`) which updates the
//!    storage buffer in place.
//! 4. Bevy's 2D render path consumes the same storage buffer through
//!    [`material::LineMaterial`] and draws an instanced quad per particle
//!    via `assets/shaders/line/render.wgsl`.
//! 5. `OnExit(AppState::Line)` runs `despawn_with::<LineRoot>` to free the
//!    entity tree. Bevy ref-counts the storage buffer + material; they're
//!    freed when the last handle drops.

pub mod compute;
pub mod material;
pub mod particle;
pub mod settings;
pub mod systems;

pub use systems::LineRoot;

use bevy::prelude::*;
use bevy::sprite::Material2dPlugin;
use wc_core::lifecycle::state::AppState;
use wc_core::settings::RegisterSketchSettingsExt;
use wc_core::sketch::{despawn_with, sketch_active};

/// Plugin that registers the Line sketch.
pub struct LinePlugin;

impl Plugin for LinePlugin {
    fn build(&self, app: &mut App) {
        app.register_sketch_settings::<settings::LineSettings>();
        app.add_plugins(Material2dPlugin::<material::LineMaterial>::default());
        app.add_plugins(compute::LineComputePlugin);

        app.add_systems(OnEnter(AppState::Line), systems::spawn_line);
        app.add_systems(OnExit(AppState::Line), despawn_with::<LineRoot>);
        app.add_systems(
            Update,
            systems::update_sim_params.run_if(sketch_active(AppState::Line)),
        );
    }
}
```
